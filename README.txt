College FAQ Chatbot Project

This project is a College FAQ Chatbot developed using Python and Natural Language Processing (NLP).

Technologies used:
- Python
- NLTK
- Tkinter

How to run:
- Use Python 3.11 or above
- Run the file: College_FAQ_Chatbot.py

The chatbot answers common college-related queries like admission, fees, courses, hostel, etc.

Thank you.
